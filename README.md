# LabelMe Quality Control

Инструмент валидации разметки LabelMe для клеток.

## Использование
1. Откройте сайт
2. Введите ваше имя
3. Загрузите JSON файл с разметкой LabelMe
4. Назначьте метки клеткам
5. Экспортируйте результаты в Excel

## Возможности
- Просмотр полного изображения и увеличенного фрагмента
- 5 типов классификации клеток
- Быстрая навигация (стрелки, горячие клавиши)
- Список всех клеток с поиском
- Экспорт в Excel
- Автосохранение в браузер
