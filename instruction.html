🔬 LabelMe Validator — Instructions

How to load JSON files, label cells, save progress, and export results

Open Tool

1) Quick Start

  1. Open index.html and enter your validator name. Your session is saved and will be restored automatically on next visit.
  2. Load annotation data using one of the following methods:
    • 📂 Load JSON — open a single LabelMe JSON file;
    • 📁 Annotations Folder — select a folder containing multiple JSON files;
    • If the tool is served via HTTP, files placed in annotations/ are listed automatically.
  3. Click a file from the list to open it and start validation.

2) How to Label Cells

  • The current cell shape is displayed in the viewer panel on the left (zoomed-in fragment of the source image).
  • Assign a label using the buttons in the Assign Label panel or keyboard shortcuts:

  • Use the number field + → to jump directly to any cell by index.
  • Use ☰ to open the full cell list with search by number, original label, or validator label.

3) Saving Progress

Automatic browser save

Progress is saved to localStorage automatically after every label assignment. It is restored when you open the tool again with the same validator name. Your session (validator name) is also preserved across page refreshes.

File-based save (recommended)

  1. Click 💾 Save Folder in the header.
  2. Choose any folder on your computer (e.g. next to your project).
  3. The tool will automatically write and update a progress file:
    <annotation_file>__<validator_name>.validation.json
  4. On the next session, select the same save folder to restore progress from the file (this takes priority over browser storage).
When running from file:// (double-clicking the HTML), automatic listing from the server is unavailable. Use the 📁 Annotations Folder button instead.

4) Exporting Results

📊 Excel report

Click 📊 Export Excel to download a .xlsx spreadsheet with a complete validation report across all files worked on by the current validator.

Columns: Validator Name, File, Cell Index, Contour Coordinates, Original Label, Validator Label, Shape Type.

📄 Corrected LabelMe JSON

Click 📄 Export JSON to download the original LabelMe JSON with the label field replaced by the corrected validator label for every validated shape. Shapes that have not been reviewed keep their original labels. The file is named <original_name>_corrected.json and is ready to use with LabelMe or downstream pipelines.

Always export your results before closing a session. Use the Excel file as a human-readable record and the corrected JSON for further processing.

5) Troubleshooting

  • JSON list does not appear — make sure you selected a folder that actually contains .json files.
  • No image, only outlines — the JSON file has no imageData field. Shapes are still visible and can be labeled.
  • Progress not restored — make sure the validator name is entered exactly as before (case-sensitive).
  • Export JSON button is disabled — a file must be loaded and at least one label must be assigned.
  • Save Folder not working — this feature requires Chrome or Edge. Firefox does not support the File System Access API.
